# German Housing Price Analysis

## Overview
This project predicts housing prices in major German cities using Machine Learning.

## Dataset Features
- City
- Size (sqm)
- Rooms
- Age of Property
- Floor
- Balcony
- Parking
- Elevator
- Distance from CBD
- District Score

## Models Used
- Linear Regression
- Ridge Regression
- Random Forest
- Gradient Boosting

## Best Model
Gradient Boosting

- R² Score: 0.8924
- MAE: €540.65 per sqm
- RMSE: €653.16 per sqm
